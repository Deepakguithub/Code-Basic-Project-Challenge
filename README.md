# Code-Basic-Project-Challenge

Retail Promotion Performance Analysis using SQL

Project Introduction

“In this project, I analyzed the impact of promotional campaigns on product sales using SQL. The objective was to evaluate how different promotional strategies such as discounts, cashback, and buy-one-get-one offers affect sales performance across different stores and product categories.”

Dataset Overview

“The dataset follows a Star Schema structure, commonly used in data warehousing and analytics systems.”

It consists of:

Fact Table

fact_evt

This table stores transactional data related to promotional events, including:

Store ID

Campaign ID

Product Code

Base Price

Promotion Type

Quantity sold before promotion

Quantity sold after promotion

Dimension Tables

dim_com (Campaign Dimension)
Contains information about marketing campaigns such as:

Campaign ID

Campaign Name

Start Date

End Date

dim_pro (Product Dimension)
Stores product details including:

Product Code

Product Name

Product Category

dim_sto (Store Dimension)
Contains store information including:

Store ID

City

Data Analysis Process
1️⃣ Sales Performance Analysis

“First, I calculated the incremental revenue generated by promotions by comparing sales before and after promotions.”

Logic:

Incremental Revenue =
(Sales After Promotion − Sales Before Promotion)

This helped measure how much additional revenue promotions generated.

2️⃣ Incremental Units Sold

“Next, I calculated the increase in quantity sold after promotions to understand which stores experienced the highest sales growth.”

This helped identify:

Best performing stores

Worst performing stores

3️⃣ Store-Level Revenue Analysis

“I joined the fact table with the store dimension table to analyze incremental revenue by city.”

This allowed insights such as:

Which cities generated the highest promotional revenue

Store-level promotional performance

4️⃣ Promotion Type Performance

“I analyzed how different promotion types performed, such as:”

25% discount

50% discount

33% discount

BOGOF (Buy One Get One Free)

Cashback promotions

For each promotion type, I calculated:

Incremental revenue

Incremental units sold

This helped determine which promotional strategy was most effective.

5️⃣ Promotion Effectiveness Score

“I created a performance score that evaluates promotion effectiveness by comparing sales lift with the average discount offered.”

This analysis helps businesses choose promotions that generate the highest return on discount investment.

6️⃣ Product Category Analysis

“Next, I analyzed which product categories benefited most from promotions.”

I calculated sales lift by category, which helped identify:

Categories with highest promotional impact

Categories with low promotional response

7️⃣ Top and Bottom Products

“I identified the top 5 and bottom 5 products based on sales lift during promotions.”

This insight helps businesses:

Promote high-performing products

Reconsider promotion strategies for low-performing products

8️⃣ Correlation Analysis

“Finally, I performed a correlation analysis between discount percentage and sales lift for each product category.”

This helped answer:

Do higher discounts lead to higher sales?

Which categories respond better to discounts?
